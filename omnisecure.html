OmniSecure — identity security that stays yours.

Zero-knowledge password manager, dev secrets, and encrypted sharing — built for OmniTender staff, partners, and developers. Self-host or run locally; your master password never leaves your device.

View on GitHub OmniAuth (2FA app)

Password Manager

Encrypted vault for logins, cards, identities, notes, and SSH keys. Sync across devices with end-to-end encryption.

Secrets Manager

Projects, service accounts, and API keys for dev teams — same zero-knowledge model as infrastructure secrets in Bitwarden.

OmniSecure Send

Share encrypted text via expiring links. Password-protected sends supported.

Browser extension

Autofill logins from your vault on any site. Load the unpacked extension from the GitHub repo.

Quick start (developers)

git clone https://github.com/subtiliorars-sys/OmniSecure.git

cd OmniSecure && pnpm install && pnpm build

pnpm dev  # API :8787 · Web vault :5173

pnpm build:extension  # Load apps/browser-extension/dist in Chrome

OmniSecure is open source (AGPL-3.0). OmniTender provides merchant payment services; OmniSecure provides identity tooling for the same ecosystem.
